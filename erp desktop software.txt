2. Realtime Database Data Wrappers
Since Realtime Database stores data as JSON trees, create helper functions to handle them:

fetchCollection(colName): Uses get(ref(db, colName)), checks snap.exists(), and returns an array of objects mapping the JSON keys to an id property.
fetchOptions(colName, labelCol): Returns { value: id, label: property } for dropdowns.
nextCode(colName, codeField, prefix, pad): Fetches a collection, maps through documents to find the highest numeric suffix in codeField, and returns the next sequential string (e.g., SO-001).
CRUD wrappers: Use push(ref(db, colName), data) for adds, update(ref(db, colName + '/' + id), data) for edits, and remove(ref(db, colName + '/' + id)) for deletes.
